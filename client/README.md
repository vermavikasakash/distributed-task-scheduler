# Distributed Task Scheduler Client

React UI for uploading task batches and monitoring the distributed scheduler.

## Features

- Upload Excel, XLS, or CSV task batches.
- Publish tasks through the API into RabbitMQ.
- Watch task status, retry count, and timestamps refresh automatically.
- Use the dashboard without user, admin, or agent access flows.

## Getting Started

```bash
npm install
npm start
```
